General settings in the console communication software

In this section, the writing of the Timax, the time attendance device, and the function of the speaker can be seen and adjusted.

Device writing: It is inserted into the system when programming the hardware.

Speaker: There are four types of sound settings in the system.

  • Full mute: In all options, the speaker is muted.
  • Mute alarms: Alarms are muted.
  • Turn off the sound of the keyboard: There is no sound when you press the keys on the keyboard.
  • Stopping the card ID recognition sound: Card registration confirmation is done without sound.

Time Settings

Setting, receiving and sending (time, date, digital or hour display format, and type of solar or Gregorian date) is available to us.

If you select the current date or time when sending information, the date and time of the computer will be sent to the device.

Main Settings of Software

In this section, we have the following settings:

Language: Determine the language of the attendance terminal in Persian or English

Datacenter: Input data to the attendance device can be transferred and used in two ways.

  • Automatically: automatically transferred to the data center.
  • Displayed as monitoring: simultaneous display of information in the monitoring software during clocking.

Fixed ID: Each Timax attendance device has a unique fixed number that can be set. This number, id, specifies the source and destination of the information for the system. For example, the information received by this device with the specified identifier is transferred to the data center with a defined fixed identifier.

Relay 1 and Relay 2: The duration of the activation of relays 1 and 2 can be programmed from 0 seconds to 255 seconds, in other words, every time the relay is activated for the time set here, the relays remain active.

Input 1 and 2: In this system, we have two TTL inputs to receive digital information, here the time of zero or one of these inputs can be programmed.

Security Settings of Attendance Device

Settings: setting the passwords of the attendance device to factory default passwords

Entering the password of the zero or MENU key or without encrypting the MENU key

Menu password error detection: By activating this option, by entering the wrong password three times, the device will lock the password and we will no longer be able to enter the menus through the device, and the password can only be changed through the software and the new password will be loaded into the menus.

Security password: entering the password of the device's security sections or without encrypting this section

Security password error detection: This option is similar to option 3 regarding the security password.
